# Sublist

Є два масиви. Перевірити чи рівні два вони, якщо ні, то перевірити чи містить перший масив всі елементи другого або навпаки.

Зокрема, список A є підписом списку B, якщо видаленням 0 або більше елементів з передньої частини B і 0 або більше елементів із задньої частини B ви отримуєте список повністю рівний A.

Приклади:

* A = [1, 2, 3], B = [1, 2, 3, 4, 5], A є sublist of B
* A = [3, 4, 5], B = [1, 2, 3, 4, 5], A є sublist of B
* A = [3, 4], B = [1, 2, 3, 4, 5], A є sublist of B
* A = [1, 2, 3], B = [1, 2, 3], A is equal to B
* A = [1, 2, 3, 4, 5], B = [2, 3, 4], A є superlist of B
* A = [1, 2, 4], B = [1, 2, 3, 4, 5];
